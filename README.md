# Resources

## List of Libraries

- python-dotenv==1.1.1
- langchain-core==0.3.72
- langchain-google-genai==2.1.8
- langchain==0.3.27
- pydantic==2.11.7
- todoist-api-python
- tzdata

In order to execute all of them in the creation of
the project you need to run the following command in the root project
console:

````
pip install -r requirements.txt
````

After that you need to create an account in the todoist app
in order to get your API Key.

[todoist app link](https://www.todoist.com/)
